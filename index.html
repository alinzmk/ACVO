WHAT IS ACVO

Active Voting (ACVO) is a decentralized ballot application what people can earn money by creating ballots and use different methods from usual. It is now at Ethereum Rinkeby test network.

Using ACVO, you can create new surveys that you can generate income from. ACVO, which provides full transparency with its structure running on the Ethereum blockchain, does not record any votes cast by users.
